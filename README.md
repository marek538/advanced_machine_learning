# 1) Fashion MNIST: Dimensionality Reduction & Binary Classification

## 📌 Project Description
This project focuses on the binary classification of high-dimensional image data (28x28 grayscale images of clothing items like T-shirts and Trousers from the Fashion MNIST dataset). 

The primary goal is to address the "curse of dimensionality" by applying and comparing various dimensionality reduction techniques and traditional machine learning algorithms to find the most accurate and efficient predictive model.
